# 模式状态文件

> 本文件记录教师助手模式的当前状态，控制是否默认启用项目上下文。

---

## 当前模式

**状态**：ON

**说明**：默认启用教师助手项目上下文，后续请求将优先参考项目记忆与规则。

---

## 模式含义

| 状态 | 含义 |
|-----|------|
| ON | 后续请求默认参考教师助手项目记忆与规则 |
| OFF | 后续请求不默认读取项目记忆，回到通用模式 |

---

## 注意事项

1. 模式开关仅影响"是否默认启用项目上下文"
2. 模式开关不等于修改模型本身
3. 模式开关不应覆盖用户当前明确指令
4. 模式开关必须可逆，用户可随时关闭
5. 即使模式为 ON，用户明确要求优先于模式默认规则
