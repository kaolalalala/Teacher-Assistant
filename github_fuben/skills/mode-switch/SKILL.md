# SKILL.md - Mode Switch（模式控制技能）

## 技能名称
mode-switch

## 技能描述
识别并执行"开启教师助手"/"关闭教师助手"口令，控制是否默认启用教师助手项目上下文。

## 触发条件

当用户输入匹配以下口令时触发：

### 开启模式口令

| 口令 | 识别为 |
|-----|-------|
| 开启教师助手 | 开启模式 |
| 打开教师助手模式 | 开启模式 |
| 进入教师助手模式 | 开启模式 |

### 关闭模式口令

| 口令 | 识别为 |
|-----|-------|
| 关闭教师助手 | 关闭模式 |
| 退出教师助手模式 | 关闭模式 |
| 关闭教师助手模式 | 关闭模式 |

## 执行规则

### 开启教师助手

```
步骤1：识别口令
   → 确认用户意图为"开启教师助手"

步骤2：更新模式状态文件
   → 读取 profiles/mode-state.md
   → 将状态更新为 ON
   → 记录切换时间和操作

步骤3：返回确认信息
   → "已进入教师助手模式"
   → "后续将默认参考教师助手项目记忆与规则"
```

### 关闭教师助手

```
步骤1：识别口令
   → 确认用户意图为"关闭教师助手"

步骤2：更新模式状态文件
   → 读取 profiles/mode-state.md
   → 将状态更新为 OFF
   → 记录切换时间和操作

步骤3：返回确认信息
   → "已退出教师助手模式"
   → "后续将不再默认读取项目记忆文件"
```

## 模式状态文件格式

### 文件路径
`profiles/mode-state.md`

### 文件结构

```markdown
# 模式状态文件

## 当前模式

**状态**：{ON/OFF}

**说明**：{模式说明文字}

## 切换记录

| 时间 | 操作 | 说明 |
|-----|------|------|
| {日期} | {操作类型} | {操作说明} |
```

## 模式行为说明

### 模式 ON 时

当模式为 ON 时，系统默认优先参考：

| 优先级 | 文件路径 | 用途 |
|-------|---------|------|
| 1 | `MEMORY.md` | 高频核心偏好 |
| 2 | `profiles/teacher-profile.md` | 教师详细偏好 |
| 3 | `profiles/classroom-profile.md` | 班级画像 |
| 4 | `profiles/defaults.md` | 系统默认配置 |

**注意**：`memory/` 目录仅在任务相关时按需读取，不每轮全量读取。

### 模式 OFF 时

当模式为 OFF 时：

- 不默认读取上述项目记忆文件
- 系统按通用模式处理
- 如用户明确要求"参考某个文件"，仍可按要求读取

## 重要说明

### 模式开关控制的是

- ✅ 是否默认启用项目上下文
- ✅ 是否默认读取项目记忆文件
- ✅ 是否默认应用教师偏好和班级画像

### 模式开关不控制

- ❌ 不修改模型本身
- ❌ 不删除记忆
- ❌ 不清空历史
- ❌ 不覆盖用户当前明确指令

### 优先级规则

```
当前用户明确要求 > 当前模式状态 > 教师偏好/班级画像 > 通用默认规则
```

## 示例

### 示例1：开启教师助手

**用户输入**：开启教师助手

**处理流程**：
1. 识别口令 → 开启模式
2. 读取 `profiles/mode-state.md`
3. 更新状态为 ON
4. 添加切换记录
5. 返回确认

**返回信息**：
```
已进入教师助手模式。
后续将默认参考教师助手项目记忆与规则。
```

### 示例2：关闭教师助手

**用户输入**：关闭教师助手

**处理流程**：
1. 识别口令 → 关闭模式
2. 读取 `profiles/mode-state.md`
3. 更新状态为 OFF
4. 添加切换记录
5. 返回确认

**返回信息**：
```
已退出教师助手模式。
后续将不再默认读取项目记忆文件。
```

### 示例3：口令变体

**用户输入**：进入教师助手模式

**处理**：识别为"开启模式"，执行开启流程

**用户输入**：退出教师助手模式

**处理**：识别为"关闭模式"，执行关闭流程

## 边界限制

### 支持

- ✅ 识别开启/关闭口令
- ✅ 更新模式状态文件
- ✅ 返回确认信息

### 不支持

- ❌ 不支持部分开启（如只开启某个模块）
- ❌ 不支持定时开关
- ❌ 不支持条件触发开关
- ❌ 不支持多用户独立模式

## 错误处理

1. **模式状态文件不存在**：创建默认文件，状态为 ON
2. **模式状态文件损坏**：重置为默认状态 ON
3. **无法识别的口令**：不触发本技能，交由主 agent 处理

---

## 边界规则

### Use when

- 用户明确要求开启或关闭教师助手模式
- 用户说"开启教师助手"、"关闭教师助手"
- 用户说"打开教师助手模式"、"退出教师助手模式"

### Do not use when

- 用户请求的是业务任务（教案/PPT/作业等）
- 用户请求的是评测或回归测试

### 冲突让位规则

本技能为**控制类**，优先级最高：

| 场景 | 行为 |
|-----|------|
| 控制类请求 | 优先拦截，不与业务类竞争 |
| 业务类请求 | 让位给对应业务类 skill |

### 控制类优先规则

- ✅ 控制类一律优先于业务类
- ✅ 不与业务类 skill 竞争
- ✅ 只响应模式开关口令
