# SKILL.md - Assignment（作业/练习技能）

## 技能名称
assignment

## 技能描述
专门负责生成作业和练习题的子技能，根据学科、年级、知识点、学段、教师偏好、班级画像等信息，输出符合学段特点和个人风格的结构化题目列表，支持课堂练习、课后作业、分层练习等多种场景。

## 触发条件
由 head-teacher 技能调用，当用户请求练习、作业、习题、巩固题、分层题、出题等内容时触发。

| 作业风格 | profiles/teacher-profile.md | 基础巩固/分层训练/提高拓展 |
| 是否偏好分层设计 | profiles/teacher-profile.md | 题目分层结构 |
| 题目来源偏好 | profiles/teacher-profile.md | 教材原题/变式题/原创 |

### 可参考班级画像

| 画像项 | 来源 | 影响内容 |
|-------|------|---------|
| 整体基础水平 | profiles/classroom-profile.md | 题目难度定位 |
| 作业完成倾向 | profiles/classroom-profile.md | 题量调整 |
| 需重点巩固的类型 | profiles/classroom-profile.md | 题目类型侧重 |
| 分层作业接受度 | profiles/classroom-profile.md | 分层设计策略 |

### 偏好应用规则

1. **优先级**：当前任务明确要求 > 教师偏好 > 班级画像 > 通用默认规则
2. **题量适配**：
   - 偏好精练 → 默认减少题量
   - 偏好标准 → 按常规数量
   - 偏好稍多 → 可适当增加
3. **班级适配**：
   - 班级基础偏弱 → 默认题目更基础，难度降低
   - 班级基础较强 → 可适度增加提高题
   - 班级作业完成需督促 → 题量适当减少
4. **分层适配**：
   - 偏好分层 + 班级分层接受度高 → 明确分层设计
   - 偏好分层 + 班级分层接受度低 → 适度分层，减少区分
5. **优雅降级**：偏好缺失时使用通用规则

### 应用示例

| 偏好组合 | 自动调整 |
|---------|---------|
| 精练 + 班级基础偏弱 | 3-4道基础题，聚焦核心 |
| 分层训练 + 班级基础较强 | 基础3题 + 提高2题 + 拓展1题 |
| 提高拓展 + 班级基础中等 | 适度提高难度，保持基础题比例 |

## 学段适配规则

### 输入参数

| 参数名 | 必需 | 说明 |
|-------|------|------|
| 学段 | 是 | 小学/初中/高中 |

### 小学练习风格

| 维度 | 要求 |
|-----|------|
| 题量 | 3-5题 |
| 题型 | 选择、填空、连线、简单问答 |
| 难度 | 基础为主，少量提高 |
| 语言 | 简单直接，生活化情境 |
| 答案 | 直接明确 |
| 时间 | 10-15分钟 |

**题目特点**：
- 题干短
- 指令清晰
- 情境生活化
- 多用图片辅助

### 初中练习风格

| 维度 | 要求 |
|-----|------|
| 题量 | 4-6题 |
| 题型 | 选择、填空、简答、计算 |
| 难度 | 基础+提高分层 |
| 语言 | 规范准确 |
| 答案 | 有解析提示 |
| 时间 | 15-25分钟 |

**题目特点**：
- 典型题目
- 考查理解与应用
- 有梯度设计

### 高中练习风格

| 维度 | 要求 |
|-----|------|
| 题量 | 4-6题（单题复杂度更高） |
| 题型 | 综合、分析、探究 |
| 难度 | 有区分度，强调能力 |
| 语言 | 专业规范 |
| 答案 | 有思路分析 |
| 时间 | 25-40分钟 |

**题目特点**：
- 综合性强
- 强调方法迁移
- 有思维深度

### 学段适配检查

| 学段 | 检查要点 |
|-----|---------|
| 小学 | 题目是否过长？语言是否简单？是否有生活化情境？ |
| 初中 | 是否有典型题？是否有梯度？难度是否适中？ |
| 高中 | 是否过于简单？是否有综合性和深度？ |

## 生成规则

### 题目数量规则

| 使用场景 | 默认数量 | 分层时数量分配 |
|---------|---------|---------------|
| 课堂练习 | 5题 | 基础3题 + 提高2题 |
| 课后作业 | 6题 | 基础3题 + 提高2题 + 拓展1题 |
| 巩固训练 | 8题 | 基础4题 + 提高3题 + 拓展1题 |

### 题型分布规则

**语文**：
- 选择题：30%
- 填空题：30%
- 简答题：40%

**数学**：
- 选择题：20%
- 填空题：30%
- 计算题/解答题：50%

**英语**：
- 选择题：40%
- 填空题：30%
- 阅读理解/写作：30%

### 难度分配规则

**标准难度**：
- 基础题：60%
- 中等题：30%
- 较难题：10%

**分层练习**：
- 基础题：难度较低，考查基本概念
- 提高题：难度中等，考查综合应用
- 拓展题：难度较高，考查创新思维

### 内容要求

1. **紧扣知识点**：题目必须围绕指定知识点设计
2. **难度递进**：从易到难排列
3. **题干清晰**：题目表述准确无歧义
4. **答案准确**：参考答案必须正确
5. **题量适中**：不超过用户指定数量

## 分层输出规则

### 分层判断

用户明确要求"分层"、"基础题和提高题"、"分层作业"时，必须分层输出。

### 分层结构

```markdown
### 基础题（必做）
- 难度：基础
- 目标：全体学生
- 数量：占总题量50-60%

### 提高题（选做）
- 难度：中等
- 目标：中等及以上学生
- 数量：占总题量30-40%

### 拓展题（挑战）
- 难度：较高
- 目标：学有余力的学生
- 数量：占总题量10-20%
```

## 边界限制

本技能：
- ✅ 生成结构化练习题
- ✅ 提供参考答案
- ✅ 支持分层输出
- ✅ 基于教案生成配套练习
- ❌ 不做自动判分
- ❌ 不声称"已布置"
- ❌ 不生成超长题单
- ❌ 不调用外部题库API
- ❌ 不做真实在线布置

## 降级规则

如果信息不足（如只说"出点题"）：
1. 自动补全默认值（学科=语文，年级=初中，数量=5题）
2. 生成通用基础版练习
3. 在输出末尾添加提示：
   > 💡 本练习基于默认设置生成。如需更精准的题目，请补充：课题、年级、难度、数量。

A. ①比喻 ②比喻、拟人 ③反复
B. ①比喻、排比 ②比喻、拟人 ③反复

## 五、参考答案

**第1题**：B
解析：A项"黄晕"应读yùn；C项"抚摸"应读fǔ；D项"润湿"应读rùn。

**第2题**：
（1）红的像火，粉的像霞，白的像雪
（2）吹面不寒杨柳风
（3）像牛毛，像花针，像细丝

**第3题**：B
解析：①运用了比喻和排比，②运用了比喻和拟人，③运用了反复和拟人。

**第4题**：
文章描绘了五幅春景图：
1. 春草图：嫩、绿、多、软
2. 春花图：花色艳丽、花味香甜
3. 春风图：柔和、芳香、悦耳
4. 春雨图：细、密、亮
5. 迎春图：人勤春早

**第5题**：
这句话运用了拟人的修辞手法。"偷偷地""钻"赋予小草人的情态，生动形象地写出了小草破土而出的情景，表现了小草旺盛的生命力，表达了作者对春天的喜爱之情。

## 统一输出规范

## 边界规则

### Use when

- 用户明确要求出题、生成练习或作业
- 用户说"出几道题"且无讲解需求
- 用户说"布置一份作业"且无辅导需求

### Do not use when

- 用户请求的是讲解或辅导（不要直接给答案）
- 用户请求的是行政文案
- 用户请求的是教案或PPT

### 冲突让位规则

| 冲突对象 | 让位条件 | 让位方向 |
|---------|---------|---------|
| tutor | 讲解需求（不要直接给答案） | 让位给 tutor |
| workflow-pack | 讲解 + 练习组合 | 让位给 workflow-pack |

### 边界判断示例

| 用户请求 | 路由目标 | 判断依据 |
|---------|---------|---------|
| "出5道练习题" | assignment | 纯出题 |
| "布置一份作业" | assignment | 纯作业 |
| "帮我讲这道题" | tutor | 讲解需求 |
| "讲解 + 出类似题" | workflow-pack | 组合请求 |

## 推荐模型档位

### 默认档位

| 属性 | 值 |
|-----|-----|
| 默认档位 | standard（标准模型） |
| 可升级 | strong |
| 可降级 | light |
| 不推荐 | 无 |

### 升级条件

满足以下条件时升级到强模型：

| 触发场景 | 判断标准 |
|----------|----------|
| 复杂分层题 | 多条件综合题型 |
| 分层作业设计 | 需要设计基础/提高/拓展多层 |
| 创新性题目 | 探究性/创新性题目设计 |
| 多条件综合 | 题目条件复杂，需要深度推理 |

### 降级条件

满足以下条件时可降级到轻量模型：

| 触发场景 | 判断标准 |
|----------|----------|
| 简单基础练习 | 简单4~6题基础练习 |
| 标准题型重复 | 标准题型，无需创新 |
| 仅需题量填充 | 只需填充题量 |

### 档位选择示例

| 用户请求 | 推荐档位 | 选择原因 |
|---------|---------|---------|
| "出5道练习题" | standard | 标准复杂度 |
| "设计分层作业，含提高拓展" | strong | 分层设计，复杂度高 |
| "简单出几道基础题" | light | 基础练习，复杂度低 |
| "创新探究题，要有深度" | strong | 创新性要求高 |
