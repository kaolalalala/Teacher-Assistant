# 预期行为基线：模式开关

## 能力名称

mode-switch（模式开关）

## 核心行为预期

### 口令识别

| 口令变体 | 识别为 |
|---------|-------|
| 开启教师助手、打开教师助手模式、进入教师助手模式 | 开启模式 |
| 关闭教师助手、退出教师助手模式、关闭教师助手模式 | 关闭模式 |

### 状态更新

- 开启模式：更新 `profiles/mode-state.md` 状态为 ON
- 关闭模式：更新 `profiles/mode-state.md` 状态为 OFF

## 模式行为差异

### 模式 ON 时

| 行为 | 说明 |
|-----|------|
| 读取 MEMORY.md | 每次任务开始时读取 |
| 读取 profiles/ | 任务开始时按需读取 |
| 参考教师偏好 | 如存在则参考 |
| 参考班级画像 | 如存在则参考 |
| 按需读取 memory/ | 任务相关时读取 |

### 模式 OFF 时

| 行为 | 说明 |
|-----|------|
| 不读取 MEMORY.md | 不默认读取 |
| 不读取 profiles/ | 不默认读取 |
| 不参考偏好 | 按通用规则执行 |
| 按明确要求读取 | 用户明确指定时可读取 |

## 禁止事项

- ❌ 口令识别错误
- ❌ 状态更新失败
- ❌ 模式 ON 时不读取项目记忆
- ❌ 模式 OFF 时默认读取项目记忆
- ❌ 模式开关覆盖用户当前明确指令

## 优先级规则

无论模式是否开启，都必须遵循：

```
当前用户明确要求 > 当前模式状态 > 教师偏好/班级画像 > 通用默认规则
```

## 模式状态文件

- **路径**：`profiles/mode-state.md`
- **内容**：记录当前模式状态（ON/OFF）
- **格式**：

```markdown
# 模式状态

## 当前状态

ON

## 状态历史

| 时间 | 操作 | 新状态 |
|-----|------|-------|
| 2026-03-01 10:00 | 开启教师助手 | ON |
```

## 验收标准

### Pass 条件

- 口令识别正确
- 状态更新正确
- ON 行为符合预期
- OFF 行为符合预期

### Partial Pass 条件

- 口令识别正确
- 状态更新正确
- 行为有轻微偏差
- 不影响核心功能

### Fail 条件

- 口令识别错误
- 状态更新失败
- ON/OFF 行为错误
- 模式开关失效
