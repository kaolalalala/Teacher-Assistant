## 覆盖场景

- [x] 核心能力：作业/练习生成

- [x] 学科：语文

---

## 预期行为

### 路由预期

- 应路由到：assignment
- 不应路由到：lesson-planner、presentation、admin、tutor

### 输出结构预期

应包含以下部分：

1. 练习标题
2. 适用对象（学科、年级）
3. 使用场景
4. 题目列表（数量符合要求）
5. 参考答案

### 行为预期

- 识别学段为"初中"
- 识别学科为"语文"
- 识别课题为"《春》"
- 识别年级为"初二"
- 题目数量为 5 道
- 输出风格应符合初中特点
