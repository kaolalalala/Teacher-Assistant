# 教师偏好模板

> 本模板定义教师偏好档案的标准结构，供系统读取和初始化使用。

---

## 一、基本信息

| 项目 | 可选值 | 默认值 |
|-----|-------|-------|
| 常教学科 | 语文/数学/英语/物理/化学/生物/历史/地理/政治/其他 | 未设置 |
| 常教年级 | 小学一年级~六年级/初中初一~初三/高中高一~高三 | 未设置 |
| 常教学段 | 小学/初中/高中 | 未设置 |

## 二、教案风格偏好

| 偏好项 | 可选值 | 默认值 |
|-------|-------|-------|
| 教案风格 | 简洁/详细/活动型/重点突破型 | 标准 |
| 是否偏爱探究式教学 | 是/否 | 否 |
| 是否更重练习设计 | 是/否 | 否 |
| 是否偏重考试导向 | 是/否 | 否 |
| 教案语言风格 | 口语化/规范化/学术化 | 规范化 |

### 各风格说明

| 风格 | 特点 |
|-----|------|
| 简洁 | 篇幅短、聚焦核心、快速备课 |
| 详细 | 步骤完整、细节丰富、便于存档 |
| 活动型 | 强调互动环节、学生参与 |
| 重点突破型 | 聚焦重难点、效率导向 |

## 偏好应用规则

### 优先级

```
当前用户明确要求 > 教师偏好 > 班级层面画像 > 通用默认规则
```

### 冲突处理

- 若当前任务明确要求与偏好不同，以当前任务要求为准
- 若偏好缺失，使用通用默认规则
- 偏好仅作为增强参考，不强制覆盖用户指令

### 应用范围

| 模块 | 可读取偏好 |
|-----|-----------|
| lesson-planner | 教案风格、语言风格、教学偏好 |
| presentation | PPT风格、文字密度、互动偏好 |
| assignment | 题量、作业风格、分层偏好 |
| tutor | 讲解风格、深度、鼓励偏好 |
| admin | 沟通语气、文案长度 |
