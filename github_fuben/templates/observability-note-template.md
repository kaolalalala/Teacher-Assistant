# 观察记录模板

> 本模板定义通用观察记录的标准结构。

---

## 基本信息

| 字段 | 内容 |
|-----|------|
| 记录日期 | {YYYY-MM-DD} |
| 记录时间 | {HH:MM} |
| 记录类型 | {任务执行/错误/降级/重写} |

---

## 任务信息

| 字段 | 内容 |
|-----|------|
| 任务类型 | {教案/PPT/作业/行政/辅导/工作流包} |
| 学段 | {小学/初中/高中} |
| 是否工作流包 | {是/否} |
| 工作流包类型 | {备课包/家校沟通包/辅导包/轻量课堂包/无} |

---

## 执行状态

| 字段 | 内容 |
|-----|------|
| 是否成功 | {是/否} |
| 是否触发降级 | {是/否} |
| 是否触发重写 | {是/否} |
| 降级原因 | {信息不足/路径缺失/偏好缺失/无} |
| 重写原因 | {风格不符/结构不完整/用户要求/无} |

---

## 模式与上下文

| 字段 | 内容 |
|-----|------|
| 模式状态 | {ON/OFF} |
| 是否读取偏好 | {是/否} |
| 是否读取班级画像 | {是/否} |
| 是否读取历史档案 | {是/否} |

---

## 备注

{简要备注，不超过50字}

---

## 记录原则

1. 只写摘要，不写完整原文
2. 不记录敏感个人信息
3. 保持轻量，便于检索
4. 日志写入不干扰主流程

---

*本模板由 observability 技能使用*
