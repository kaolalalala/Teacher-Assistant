## 使用说明

1. 复制本模板到目标目录
2. 重命名为规范格式（如 `2026-03-01.md`）
3. 填写内容
4. 删除本使用说明

### 各类型命名建议

| 类型 | 目录 | 命名格式 |
|-----|------|---------|
| daily | memory/daily/ | `YYYY-MM-DD.md` |
| weekly | memory/weekly/ | `YYYY-WXX.md` |
| lesson | memory/lessons/ | `{学科}_{课题}_经验.md` |
| class | memory/classes/ | `{班级标识}_{时间段}.md` |
| pack | memory/packs/ | `{包类型}_经验.md` |
| preference | memory/preferences-history/ | `preferences-change_YYYY-MM-DD.md` |
