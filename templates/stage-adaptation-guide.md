# 学段适配指南

## 概述

本指南定义了系统在不同学段（小学/初中/高中）下的输出风格差异，确保各模块输出内容符合对应学段的教学特点和学生认知水平。

---

## 一、学段分类

| 学段 | 年级范围 | 年龄段 | 认知特点 |
|-----|---------|-------|---------|
| 小学 | 一年级~六年级 | 6-12岁 | 具体形象思维为主，注意力时间短，需要趣味引导 |
| 初中 | 七年级~九年级（初一~初三） | 12-15岁 | 抽象思维发展，理解力增强，需要结构化学习 |
| 高中 | 高一~高三 | 15-18岁 | 抽象思维成熟，逻辑性强，需要深度分析和迁移 |

---

## 二、学段识别规则

### 年级关键词识别

| 学段 | 关键词 |
|-----|-------|
| 小学 | 小学、一年级、二年级、三年级、四年级、五年级、六年级、1年级、2年级... |
| 初中 | 初中、初一、初二、初三、七年级、八年级、九年级、7年级、8年级、9年级 |
| 高中 | 高中、高一、高二、高三、十年级、十一年级、十二年级 |

### 默认策略

- 用户明确提供学段/年级 → 使用对应学段风格
- 用户未提供 → 默认使用"初中"风格（通用性最强）
- 无法判断 → 生成通用版，提示可补充学段优化

## 三、教案风格差异

### 小学教案

| 维度 | 特点 |
|-----|------|
| 整体风格 | 活泼、趣味、生活化 |
| 语言表达 | 口语化、简单易懂、多用比喻 |
| 导入设计 | 故事、游戏、情境导入为主 |
| 活动设计 | 具体可操作、时间短、互动多 |
| 环节时长 | 每环节5-10分钟，节奏快 |
| 板书设计 | 图文并茂、色彩丰富 |
| 评价方式 | 鼓励性为主，多用表扬 |
| 作业设计 | 少量、有趣、实践性强 |

**示例特点**：
- 教学目标表述简单直接
- 教学流程多用"小朋友们"
- 活动设计具体到"谁做什么说什么"
- 强调"趣味性"和"参与度"

### 初中教案

| 维度 | 特点 |
|-----|------|
| 整体风格 | 结构清晰、逻辑性强 |
| 语言表达 | 规范准确、适度专业 |
| 导入设计 | 问题导入、情境导入、复习导入 |
| 活动设计 | 有明确目标、时间适中 |
| 环节时长 | 每环节10-15分钟，节奏适中 |
| 板书设计 | 条理清晰、重点突出 |
| 评价方式 | 过程与结果并重 |
| 作业设计 | 分层设计、典型题目 |

**示例特点**：
- 教学目标分三维表述
- 教学流程结构化
- 重难点明确
- 强调"理解"和"应用"

### 高中教案

| 维度 | 特点 |
|-----|------|
| 整体风格 | 紧凑高效、深度分析 |
| 语言表达 | 专业规范、逻辑严密 |
| 导入设计 | 直奔主题、问题驱动 |
| 活动设计 | 思维训练为主、探究性强 |
| 环节时长 | 每环节15-20分钟，深度展开 |
| 板书设计 | 知识框架、思维导图 |
| 评价方式 | 能力导向、方法评价 |
| 作业设计 | 综合性强、有区分度 |

**示例特点**：
- 教学目标聚焦核心素养
- 教学流程强调思维过程
- 重难点分析深入
- 强调"迁移"和"方法"

## 四、PPT/课件风格差异

### 小学课件

| 维度 | 要求 |
|-----|------|
| 文字密度 | 每页不超过3-5行 |
| 句子长度 | 短句为主，每句不超过15字 |
| 配色风格 | 明亮、活泼、多彩 |
| 图片使用 | 大图为主，吸引注意力 |
| 动画效果 | 可适当丰富，增加趣味 |
| 互动设计 | 提问、游戏、操作环节多 |
| 页面节奏 | 一页一个重点，节奏快 |

**禁忌**：
- ❌ 满页文字
- ❌ 抽象概念过多
- ❌ 长篇讲解

### 初中课件

| 维度 | 要求 |
|-----|------|
| 文字密度 | 每页5-8行 |
| 句子长度 | 适中，表达完整 |
| 配色风格 | 清爽、专业、统一 |
| 图片使用 | 图文结合，辅助理解 |
| 动画效果 | 适度，服务于内容 |
| 互动设计 | 有提问、讨论、练习 |
| 页面节奏 | 知识点+例题+练习 |

**特点**：
- 结构清晰
- 重点突出
- 典型例题

### 高中课件

| 维度 | 要求 |
|-----|------|
| 文字密度 | 可适当提高，但保持清晰 |
| 句子长度 | 完整表达，逻辑清晰 |
| 配色风格 | 简洁、专业、沉稳 |
| 图片使用 | 图表、思维导图为主 |
| 动画效果 | 简洁，突出逻辑 |
| 互动设计 | 问题探究、思路分析 |
| 页面节奏 | 知识框架+方法总结+题型归纳 |

**禁忌**：
- ❌ 过度卡通化
- ❌ 过于简单幼稚
- ❌ 缺乏深度

## 五、作业/练习风格差异

### 小学练习

| 维度 | 要求 |
|-----|------|
| 题量 | 3-5题 |
| 题型 | 选择、填空、连线、简单问答 |
| 难度 | 基础为主，少量提高 |
| 语言 | 简单直接，生活化情境 |
| 答案 | 直接明确 |
| 时间 | 10-15分钟 |

**题目特点**：
- 题干短
- 指令清晰
- 情境生活化
- 多用图片辅助

### 初中练习

| 维度 | 要求 |
|-----|------|
| 题量 | 4-6题 |
| 题型 | 选择、填空、简答、计算 |
| 难度 | 基础+提高分层 |
| 语言 | 规范准确 |
| 答案 | 有解析提示 |
| 时间 | 15-25分钟 |

**题目特点**：
- 典型题目
- 考查理解与应用
- 有梯度设计

### 高中练习

| 维度 | 要求 |
|-----|------|
| 题量 | 4-6题（单题复杂度更高） |
| 题型 | 综合、分析、探究 |
| 难度 | 有区分度，强调能力 |
| 语言 | 专业规范 |
| 答案 | 有思路分析 |
| 时间 | 25-40分钟 |

**题目特点**：
- 综合性强
- 强调方法迁移
- 有思维深度

## 六、辅导讲解风格差异

### 小学辅导

| 维度 | 要求 |
|-----|------|
| 讲解步骤 | 更细，每步更小 |
| 语言风格 | 温和、鼓励、简单 |
| 引导方式 | 多用提问，少用术语 |
| 鼓励频率 | 高，每步都有肯定 |
| 类比使用 | 多用生活化比喻 |

**示例表达**：
- "真棒！我们再来看下一步..."
- "想一想，这像不像我们平时..."
- "你试试看，没关系的"

### 初中辅导

| 维度 | 要求 |
|-----|------|
| 讲解步骤 | 适中，逻辑清晰 |
| 语言风格 | 规范、准确 |
| 引导方式 | 问题引导+方法点拨 |
| 鼓励频率 | 适度 |
| 类比使用 | 适当使用 |

**示例表达**：
- "这个问题的关键是..."
- "我们来看看解题步骤..."
- "这类题的思路是..."

### 高中辅导

| 维度 | 要求 |
|-----|------|
| 讲解步骤 | 强调思路，不过度细碎 |
| 语言风格 | 专业、精炼 |
| 引导方式 | 思路分析+方法归纳 |
| 鼓励频率 | 适度，侧重能力肯定 |
| 类比使用 | 强调知识迁移 |

**示例表达**：
- "这道题考查的核心是..."
- "解题的关键思路..."
- "这类问题的通法是..."

## 七、行政文案风格差异

### 小学场景

| 维度 | 特点 |
|-----|------|
| 语气 | 温和、亲切、鼓励 |
| 称呼 | "各位家长朋友"、"亲爱的家长" |
| 重点 | 习惯培养、兴趣激发、家校配合 |
| 表达 | 更口语化、生活化 |

**示例**：
> 亲爱的家长朋友们，本周孩子们在课堂上表现非常棒！...

### 初中场景

| 维度 | 特点 |
|-----|------|
| 语气 | 平和、专业、诚恳 |
| 称呼 | "各位家长" |
| 重点 | 学习状态、习惯养成、阶段目标 |
| 表达 | 规范、清晰 |

**示例**：
> 各位家长：本周班级学习情况总结如下...

### 高中场景

| 维度 | 特点 |
|-----|------|
| 语气 | 简洁、专业、目标导向 |
| 称呼 | "各位家长" |
| 重点 | 学习效率、目标管理、考试准备 |
| 表达 | 精炼、重点突出 |

**示例**：
> 各位家长：现将本阶段学习情况反馈如下...

## 八、降级策略

### 学段不明时

1. **默认使用初中风格**：通用性最强，适配面最广
2. **输出末尾添加提示**：
   > 💡 本内容采用通用风格生成。如需更精准适配，请补充"小学/初中/高中"信息。

### 风格不匹配时

若检测到输出风格与学段不符：
- 小学输出过于抽象/密集 → 触发简化
- 高中输出过于幼态/简单 → 触发深化
- 初中输出失衡 → 触发平衡调整
