# 预期行为基线目录

本目录存放"预期行为基线说明"，用于对照检查系统行为是否正确。

## 重要说明

**这里不是存"标准答案原文"，而是存"预期结构和行为标准"。**

例如：

- 教案应包含 6 个部分
- 备课包应按 教案 → PPT → 练习 输出
- 模式 ON 时应默认参考偏好
- 模式 OFF 时不默认读取项目记忆

## 基线文件命名

- `baseline-[能力类型].md`
- 例如：`baseline-lesson-plan.md`

## 基线内容结构

每份基线应包含：

1. **能力名称**：对应哪个模块
2. **核心行为预期**：应该做什么
3. **输出结构要求**：必须包含哪些部分
4. **禁止事项**：不应该发生什么
5. **模式相关行为**：ON/OFF 下的差异

## 基线使用方式

1. 执行测试用例
2. 对照基线检查行为
3. 判断是否满足预期
4. 记录偏差点

## 注意事项

- 基线应保持稳定
- 基线更新需要明确记录
- 基线不是逐字参考，而是行为标准
