# 评测与回归测试目录

本目录用于存放教师助手系统的半自动评测与回归测试相关文件。

## 目录结构

```
eval/
├── cases/       # 固定测试用例
├── results/     # 评测结果记录
└── baselines/   # 预期行为基线说明
```

## 核心原则

1. **不逐字比对**：评测标准不是"和参考文本完全一致"
2. **行为优先**：重点检查路由、结构、顺序、模式是否正确
3. **允许变化**：措辞轻微变化不影响判定
4. **结果可追溯**：每轮评测都留下结果文件

## 评测结果分类

| 结果 | 说明 |
|------|------|
| Pass | 核心行为符合预期，输出结构完整，无明显退化 |
| Partial Pass | 能完成任务，但结构/风格/顺序有轻微偏差 |
| Fail | 路由错误、结构缺失严重、模式失效、核心功能不可用 |

## 使用场景

- 更新了 `AGENTS.md` 或某个 `SKILL.md`
- 调整了模式开关规则
- 调整了输出结构
- 新增了重要模板
- 需要验证"这次修改是否破坏已有能力"
