## 预期行为

- 应识别为：备课包工作流

1. 教案（完整结构）
2. PPT 课件（逐页内容）
3. 课堂练习（题目+答案）

### 执行顺序预期

```
lesson-planner → presentation → assignment
```

### 行为预期

- 识别为备课包工作流
- 识别学段为"初中"
- 识别学科为"语文"
- 识别课题为"《春》"

- 所有输出使用相同学段风格
- 整合输出，风格统一
