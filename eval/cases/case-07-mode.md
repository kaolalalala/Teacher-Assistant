## 预期行为

### 子测试 A：关闭教师助手

#### 路由预期

- 应识别为：模式切换口令
- 应调用：mode-switch 技能

#### 行为预期

- 识别为模式关闭口令
- 更新 `profiles/mode-state.md` 状态为 OFF
- 返回确认信息

### 子测试 B：开启教师助手

#### 路由预期

- 应识别为：模式切换口令
- 应调用：mode-switch 技能

#### 行为预期

- 识别为模式开启口令
- 更新 `profiles/mode-state.md` 状态为 ON
- 返回确认信息

## 模式行为验证

### 验证模式 OFF 行为

在模式 OFF 状态下执行一个教案任务：

```
帮我备一节《春》的课
```

预期：
- 不默认读取项目记忆文件
- 按通用规则执行

### 验证模式 ON 行为

在模式 ON 状态下执行一个教案任务：

```
帮我备一节《春》的课
```

预期：
- 默认读取 MEMORY.md
- 按需读取 profiles/ 目录
- 参考教师偏好（如存在）

## 备注

本用例为模式开关的基础测试用例，验证模式切换和上下文门控能力。

测试顺序：

5. 验证 ON 状态行为
