# 系统默认配置

> 本文件存放系统级默认规则，当教师偏好和班级画像缺失时使用。
> 这些是"降级策略"，确保系统在信息不足时仍能正常输出。

---

## 一、学段不明时的默认策略

| 场景 | 默认策略 |
|-----|---------|
| 学段未指定 | 使用初中标准 |
| 年级未指定 | 使用该学段中间年级 |
| 学科未指定 | 使用语文默认格式 |

---

## 二、教案默认规则

| 项目 | 默认值 |
|-----|-------|
| 教案风格 | 标准结构化 |
| 教学环节 | 导入→新授→练习→小结→作业 |
| 语言风格 | 规范化 |
| 时间分配 | 40分钟课堂标准 |

---

## 三、PPT 默认规则

| 项目 | 默认值 |
|-----|-------|
| PPT风格 | 重点突出型 |
| 每页内容量 | 标准 |
| 文字密度 | 适中 |
| 页数估算 | 每环节2-3页 |

---

## 四、作业/练习默认规则

| 项目 | 默认值 |
|-----|-------|
| 题量 | 标准（8-12题） |
| 题型分布 | 基础60% + 提高30% + 拓展10% |
| 分层设计 | 不分层（除非明确要求） |
| 题目来源 | 教材变式为主 |

---

## 五、辅导讲解默认规则

| 项目 | 默认值 |
|-----|-------|
| 讲解风格 | 分步细讲 |
| 讲解深度 | 适中 |
| 鼓励性语言 | 适度使用 |
| 举一反三 | 是 |

---

## 六、行政文案默认规则

| 项目 | 默认值 |
|-----|-------|
| 通知语气 | 正式温和 |
| 文案长度 | 标准 |
| 结构 | 标题+正文+落款 |

---

## 七、工作流包默认规则

| 项目 | 默认值 |
|-----|-------|
| 备课包内容 | 教案+PPT+练习 |
| 辅导包内容 | 讲解+练习 |
| 家校沟通包内容 | 通知+反馈建议 |

---

## 使用说明

1. **触发条件**：当 `profiles/teacher-profile.md` 或 `profiles/classroom-profile.md` 中对应字段为空时
2. **优先级**：当前用户要求 > 教师偏好 > 班级画像 > 本文件默认值
3. **编辑建议**：如需修改默认值，请直接编辑本文件
